# Strong form PINNs framework for multi-cavity mechanics
Codes and data for the paper “Why Strong-Form PINNs Tend to Fail in the Mechanics of Underground Cavities” (currently under review).

### For the purpose of promoting broader application of PINNs, the relevant code and FEM validation dataset associated with the proposed strong-form PINNs framework will be made publicly available within one month of the paper's online publication.
